Read the exact versioned docs at https://docs.expo.dev/versions/v56.0.0/ before writing any code.

Use Bun as the package manager for this repo. Prefer `bun install`, `bun add`, `bun remove`, and `bun run` instead of npm, pnpm, or yarn commands.

Do not start dev servers from an agent session. Do not run commands such as `bun start`, `bun run start`, `expo start`, `convex dev`, or any other long-running local server unless the user explicitly overrides this instruction.
